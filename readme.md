# CRUD Inventory App

Este é um aplicativo CRUD (Create, Read, Update, Delete) para gerenciamento de inventário de produtos, desenvolvido como parte do processo seletivo para a Escola Politécnica | Centro de Inovação - PUCRS, em parceria com a Aceleradora Ágil.

## Tecnologias Utilizadas

Este projeto foi construído utilizando as seguintes tecnologias:

*   **Frontend:** HTML, CSS, JavaScript
*   **Backend:** Node.js com Express
*   **Banco de Dados:** Arquivo JSON (simples armazenamento de dados)
*   **Outras Bibliotecas:** Cors

## Como Rodar a Aplicação Localmente

1.  **Clone o repositório:**
    `git clone https://github.com/ronaldodeschain/crud-js.git`
2.  **Navegue até o diretório do projeto:**
    `cd crud-js`
3.  **Instale as dependências do backend:**
    `npm install`
4.  **Inicie o servidor backend:**
    `npm start`
5.  **Utilize o Live Server para rodar a aplicação e abra o `index.html` no seu navegador através do Live Server.**
